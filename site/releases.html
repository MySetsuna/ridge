v0.1.0 "开荒 · Breaking Ground"
FIRST RELEASE 2026-04-30 在 GitHub 上查看 →

Ridge 的第一犁。把田埂铺到位、把 PTY 跑起来、把 Agent 接上、把零警告闸门拉上。

核心 · 终端

  • portable-pty + xterm.js (WebGL) 终端实现，Unicode 11、超链接、search 全部接入
  • 分块滚回：64 KiB 块、4 MiB 总上限、按 seq 字节计数器分页 (get_pane_scrollback_tail / _before)
  • Pane 标题栏显示前台进程名 + cwd（多色），IME 合成防护
  • OSC 7 cwd 同步；Windows 路径强制 forward-slash 标准化（前后端一致）

核心 · 分屏 / 工作区

  • 递归分屏树（horizontal / vertical / 嵌套），快捷键 + 鼠标拖拽 resize
  • 多工作区，独立 PTY 命名空间；非活动工作区保活
  • 资源管理器两级树（工作区 → 终端），文件树拖拽 / 重命名 / 键盘导航
  • 侧边栏可折叠 + 快捷键

核心 · 编辑器

  • Monaco 编辑器作为另一种 Pane 模式，与终端共用同一棵 PaneTree
  • 编辑器字体下拉框；三套主题
  • 侧栏 Search 跨工作区并行扫描所有 cwd（去重 + glob 过滤 + 可选 replace）

核心 · Git / SCM

  • Canvas Git Graph 可视化
  • 每个 Pane 标题栏的 PaneGitPill：分支 + diff 摘要 + 内嵌分支选择 / 创建
  • SCM 状态由 .git/ 文件监听驱动，无轮询；linked worktree 自动识别
  • round 64 SCM 刷新策略：仅两条触发路径（cwd 变 + watcher 事件），mount 时只在缓存陈旧 (>30s) 时拉取

核心 · Agent 团队

  • 内置自研 tmux shim 二进制（src-tauri/src/bin/tmux.rs），让 Claude Code 把 Ridge 当成 tmux 会话
  • shim 支持 list-panes (含 cwd) / display-message 模板变量 / kill-pane / rename-window
  • tmux shim 随安装包发布，PATH 注册由 NSIS / WiX 处理
  • RIDGE_TEAMMATE_URL/TOKEN 自动注入到 PTY 子进程

UI / 体验

  • WindDialog API：所有 alert / confirm / choice / prompt 走自定义模态，避免 OS chrome 破坏视觉一致性
  • overlayscrollbars 接管复杂滚动区，水平 tab 滚动专用 preset
  • Modal z-index 寄存表，所有浮层挂在根布局之外避免父级 stacking-context
  • 滚回历史模态：256 KiB 初拉 + 128 KiB 分页加载更早，ANSI 剥离用于复制 / 搜索

工程闸门

  • cargo build --lib 零警告，CI 可安全跑 cargo clippy -- -D warnings
  • Future-use API 与兼容 stub 用 #[allow(dead_code)] + 一行说明显式标注
